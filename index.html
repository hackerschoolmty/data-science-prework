Data Science Prework POR HACKER SCHOOL MTY

Conocimiento técnico previo requerido

  • Conocimiento básico de programación orientada a objetos en al menos un lenguaje de programación (de preferencia Python 3.x):
    • Variables y Tipos
    • Listas
    • Diccionarios
    • Operadores básicos
    • Operaciones con Strings
    • Condicionales
    • Bucles
    • Funciones
    • Clases y Objetos
    • Recursos
      • https://www.codecademy.com/learn/python
  • Conocimiento de matemáticas y habilidades para resolver problemas
  • Conocimiento básico de estadística
    • Probabilidad
    • Estadística descriptiva
    • Recursos
      • https://www.khanacademy.org/math/probability

Instalación de software necesario

  • Laptop (Con cualquier sistema operativo, pero se recomienda que sea basado en UNIX)
  • Dentro del curso se proveerán las herramientas y tutoriales para instalar Python 3.x con los siguientes paquetes:
    • Numpy
    • Scipy
    • Statsmodels
    • Matplotlib
    • Scikit-learn
    • Pandas
Puedes Instalar estas liberías de dos maneras: Usando Anaconda siguiendo las instrucciones. O si prefieres, instalar de manera directa los paquetes necesarios
